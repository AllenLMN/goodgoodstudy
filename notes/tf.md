## 1. 简介  
### 1.1 tensorflow是什么
  google的开源软件库，官网：https://tensorflow.google.cn/  
+ 采用数据流图（节点：处理数据，线：节点间的输入输出关系，线上运输张量，节点被分配到各种计算设备上运行），用于数值计算
+ 支持多平台：GPU、CPU、移动设备
+ 最初用于深度学习，目前越来越通用  

特性  
+ 高度的灵活性
+ 真正的可移植性
+ 产品和科研结合
+ 自动求微分
+ 多语言支持
+ 性能最优化
